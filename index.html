Gridsystem & Components Styles

Grid System is a grid system similar to Bootstrap, which allows you to organize the content of your website so that it can be adapted to different devices.

You can collaborate

If you are attracted by this small project, you can collaborate to make GC & CS grow.

Let's see what contains GC & CS and encourage you to use it and improve it.

Features

This system works by means of rows and columns, using a system similar to the bootstrap (12 grids). To organize the content of the website.

This system is responsive, allowing it to adapt to different devices.

The "container" class must be used to contain the "row" classes.

The "row" class must contain the "col" classes.

The "col" class must be accompanied by a second class which indicates the number of grids to be covered. These classes that indicate the amount of grids to be covered are: l - # (large devices), m - # (medium devices), s - # (small devices) and xs (extra small devices). Where "#" indicates the number of columns to be covered, this number is given by a number from 0 to 12, where 0 does not include any columns and 12 covers the entire width of its container

NOTE: the 0 indicates that at this point the column will not be visible.
